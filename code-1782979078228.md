# Quick Start Guide

Get your local instance of Lucy EMMA up and running in under five minutes.

## Prerequisites

Before starting, ensure you have the following installed on your local host:
- **Python 3.9+**
- **Node.js 18+** & **npm**
- **Git**

---

## 1. Clone the Repository

```bash
git clone [https://github.com/scottymicfree/lucy_emma_studios.git](https://github.com/scottymicfree/lucy_emma_studios.git)
cd lucy_emma_studios
